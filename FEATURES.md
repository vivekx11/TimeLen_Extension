# TimeLens Features Guide

## рҹҺҜ Complete Feature List

### 1. Real-Time Tracking

**What it does:**
- Automatically tracks time spent on each website
- Groups by domain (e.g., all YouTube pages counted together)
- Updates every 10 seconds
- Pauses when you're idle (60+ seconds)

**How it works:**
```
You visit youtube.com вҶ’ Timer starts
You switch to github.com вҶ’ YouTube timer stops, GitHub timer starts
You minimize browser вҶ’ All timers pause
You return вҶ’ Timer resumes on active tab
```

**Benefits:**
- No manual input required
- Accurate to the second
- Works across all tabs and windows
- Respects your privacy

### 4. Focus Mode

**What it does:**

2. Click "Manage Limits"
3. Add websites with time limits:
   ```

   twitter.com вҶ’ 30 minutes

**What happens:**
```
